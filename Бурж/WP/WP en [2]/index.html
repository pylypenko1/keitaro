    Cookie Settings

    Cookies are small pieces of data sent from a website and stored in a user's web browser while the user is browsing. Each cookie is saved by the browser in a small file called a cookie file. When the user requests another page from the server, the browser sends the cookie file back to the server. Cookies were designed as a reliable mechanism for websites to remember information or to track the user's browsing activity.

      Necessary ∨

    Strictly necessary cookies - typically these cookies are necessary for the website to function. They can help show the user relevant information, customize site operation, and ensure security. Usually, without these cookies, the site's operation becomes impossible or greatly restricted.

      Functional ∨

    Functional cookies - typically used to enhance site functionality and ease of use. These cookies help remember user settings (e.g., language and time zone). They allow users to avoid changing settings with each site visit.

      Statistics ∨

    Statistics cookies - typically these cookies show if the user has visited your site before. They allow tracking the number of users and their activity on the site.

      Marketing ∨

    Marketing, targeting, and advertising cookies - typically used to track user visits, viewed pages, and link clicks. These cookies help display ads relevant to the user's interests. They also help determine if the user has seen specific ads.
